Setting up the "Size" tab 8

The size of the bubbles determines by the field assigned for the "Size" tab. The higher the value, the bigger the size of the bubbles.


Steps: 

  1. Select "Make" column from the table in the Field pane.  Drag and Drop to the "Size" tab.
  2. Set "Count" for aggregation. ( by default it's set to the "Count" aggregate)
